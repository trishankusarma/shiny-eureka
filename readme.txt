To start the Project :
    1. Go to the root directory where the Project is located
    
    2. To run the backend
            
           >  cd back-end
           >  npm i
           >  npm run devStart

    3. To run the frontend
            
           >  cd front-end
           >  npm i
           >  npm start


This project deals with a full stack application for Exam management portal
